# Advanced Stock Price Checker

This project is an advanced stock price checker built with NestJS. It fetches stock prices from the Finnhub API, calculates moving averages, and stores the data in PostgreSQL. It also includes a scheduled checker using node-cron that runs at one-minute intervals to update stock prices.

## Features

- Fetch stock prices from the Finnhub API
- Calculate moving averages for stock prices
- Store stock prices in a database
- Scheduled checker to update stock prices at regular intervals
- OpenAPI documentation with Swagger

## Project setup

```bash
$ npm install
```

Rename the `.env.example` to `.env`, add the Finnhub.io API key, and change the database connection options

```
STOCK_MARKET_API_KEY=your_finnhub_api_key
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Access the OpenAPI documentation

You can access the OpenAPI documentation generated by Swagger at:
```
localhost:3000/api
```
